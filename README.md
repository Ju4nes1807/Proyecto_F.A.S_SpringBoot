# Plateamiento del problema

En Bogotá, las escuelas de fútbol comunitarias que operan en parques públicos carecen de sistemas para organizar eficientemente sus entrenamientos y torneos. Estas iniciativas, lideradas por entrenadores independientes sin estructura institucional formal, gestionan sus actividades mediante métodos tradicionales: cuadernos físicos, grupos de WhatsApp saturados, llamadas telefónicas y listas en papel.

# Justificación

Se desarrolla un software accesible y funcional que facilite la organización de entrenamientos y torneos para las escuelas de fútbol comunitarias que operan en parques públicos de Bogotá. Esta herramienta digital permitirá a entrenadores independientes y líderes deportivos barriales mejorar significativamente su gestión deportiva, reemplazando métodos rudimentarios como cuadernos físicos, papeleo innecesario y el uso disperso y desorganizado de grupos de WhatsApp.

# Objetivo General

Desarrollar Football  Association  System, un software que permita a líderes de escuelas de fútbol comunitarias en Bogotá registrar y gestionar sus escuelas y jugadores de manera digital, mientras administradores del sistema organizan torneos entre las escuelas registradas y coordinan el uso de espacios deportivos públicos. El sistema busca mejorar la planificación de entrenamientos y fomentar la participación en competencias organizadas de manera eficiente y estructurada.

# Objetivos Especificos

* Registro y Gestión de Escuelas: Permitir a líderes inscribir sus escuelas registrando nombre, ubicación, categorías y datos del responsable para facilitar su participación en entrenamientos y torneos.

* Gestión de Jugadores: Facilitar el registro de jugadores por cada líder, incluyendo nombre, edad, categoría y contacto, manteniendo un listado actualizado de participantes.

* Coordinación de Espacios Deportivos: Implementar un módulo administrativo para registrar y gestionar espacios públicos disponibles, facilitando su asignación para entrenamientos y partidos.

* Planificación de Entrenamientos: Permitir a líderes crear y gestionar entrenamientos programando fechas, horarios, espacios y categorías, con control de sesiones realizadas.

* Organización de Torneos: Desarrollar un sistema para que administradores creen torneos, agrupen escuelas por categoría, definan calendarios y se asignen los correspondientes espacios,
